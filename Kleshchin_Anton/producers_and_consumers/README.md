# Producers and consumers

# Requirements
<code>g++</code> at least 7th version and <code>make</code>.

# How to build
<code>make</code>

# Usage
<code>./run \<amount of produsers\> \<amount of consumers\></code>

# Examples
<code>./run 42 27</code>

And after a few second press `enter` to finish the test.
